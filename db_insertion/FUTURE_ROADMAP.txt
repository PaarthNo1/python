--------------------------------------------------------------------------------
**Goal:** User ko natural language mein search karne ki power dena.
**Example Query:** "Find floats in the Indian Ocean that measured low oxygen levels in 2024."

**Tasks:**
1.  **Text Generation:** Har float aur profile ka ek "Description" banana hoga (e.g., "Float 123 is an APEX float deployed by India...").
2.  **Embedding Generation:** OpenAI ya Open Source model (HuggingFace) ka use karke in descriptions ko "Vectors" (Numbers) mein convert karna.
3.  **pgvector Setup:** PostgreSQL mein `pgvector` extension enable karna taaki wo vectors store kar sake.
4.  **Semantic Search Script:** Ek script jo user ki query ko vector mein badle aur database se "Nearest Neighbors" dhoondh kar laye.
